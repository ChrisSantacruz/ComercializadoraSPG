# Frontend Redesign Prompt

Actúa como un senior frontend engineer especializado en productos SaaS modernos.

Objetivo:
rediseñar esta interfaz para que se vea premium y profesional.

Restricciones:
- mantener React

- reutilizar componentes existentes

3. proponer nueva estructura
