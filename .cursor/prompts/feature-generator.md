# Feature Generator Prompt

Actúa como senior fullstack engineer MERN.

Objetivo:
crear una nueva feature manteniendo consistencia total con el proyecto.

Antes de generar:
1. analizar arquitectura existente
2. detectar patrones actuales
3. reutilizar componentes existentes
4. reutilizar servicios existentes

La feature debe:
- ser modular
- escalable
- limpia
- responsive
- segura
- mantenible

Evitar:
- duplicación
- hacks rápidos
- código innecesario
